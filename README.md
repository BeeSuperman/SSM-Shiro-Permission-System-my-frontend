## 📸 功能展示

|---|---|
| 登入頁 | 支援帳號密碼輸入，點擊登入跳轉主頁 |

## 🛠 完整專案技術棧

- **後端**：Java Spring MVC 5 + MyBatis + Apache Shiro
- **前端**：JSP + jQuery EasyUI
- **資料庫**：MySQL 8
- **部署**：Tomcat 8.5

## 📌 完整版說明

完整版需要在本機安裝 JDK 8 + Tomcat + MySQL 才能運行。
